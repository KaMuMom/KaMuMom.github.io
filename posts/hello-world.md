# Hello World - 我的第一篇博客

哈喽！欢迎来到小鱼干的个人博客！这是我人生中的第一篇博客文章，想想还有点小激动呢。今天想跟大家聊聊我为什么要搞这个博客，以及折腾这个博客的过程。

## 为什么要写博客？

说实话，在这个短视频横行的时代，写博客听起来确实有点"老土"。但我还是觉得文字有它独特的魅力，对我来说写博客有这么几个好处：

### 1. 整理思路
我发现通过写作，能把自己脑子里的想法整理得更清楚。有时候学了一个新东西，觉得自己懂了，但一要写出来就发现其实还有很多地方没搞明白。写作逼着我把知识点重新梳理一遍，理解也会更深。

### 2. 分享交流
技术这东西嘛，就是要多交流多分享。我把自己的学习心得、踩过的坑都写出来，说不定能帮到其他朋友。而且大家也可以在评论区讨论讨论，互相学习，多好！

### 3. 逼自己进步
坚持写博客其实是对自己的一个挑战。它不仅锻炼了我的表达能力，更重要的是让我养成了持续学习的习惯。毕竟要写出有价值的内容，自己就得先学明白才行。

## 技术栈

这个博客用了下面这些技术：

- **Next.js 14**: 一个超棒的 React 框架，可以把网站生成成静态页面
- **Tailwind CSS**: 写 CSS 特别爽的工具，不用写一堆自定义样式
- **TypeScript**: JavaScript 的加强版，能帮我少踩很多坑
- **GitHub Pages**: GitHub 提供的免费网站托管，学生党福音

选这些技术主要是因为它们都比较成熟，遇到问题容易找到解决方案，而且做出来的网站速度很快。

## 未来计划

关于这个博客，我有几个小目标：

1. **保持更新**: 争取每个月至少写一篇技术文章
2. **系列文章**: 找几个感兴趣的技术点深入写写，做成系列
3. **项目分享**: 把自己做的一些小项目拿出来跟大家分享
4. **学习记录**: 记录学习新技术时的心得和踩坑经历

## 写在最后

感谢你看到这里！这是我的第一篇博客，如果有什么写得不好的地方还请多多包涵。

如果你有什么想法或者想要交流的话题，随时可以联系我哦！

希望这个博客能记录我的成长历程，也能给其他朋友带来一些帮助。让我们一起加油，一起进步！💪
